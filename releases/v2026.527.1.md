# v2026.527.1

> Released: 2026-05-27

Patch release fixing a packaging bug that prevented the published CLI from starting.

## Fixes

- **CLI launches from npm again** — The published `@dealdesk/server` package did not expose its `./cost-backfill` subpath in `publishConfig.exports`, so the bundled `dealdesk` CLI crashed on startup with `ERR_PACKAGE_PATH_NOT_EXPORTED`. The subpath is now exported from the compiled output, so `npx dealdesk@latest onboard` works.

## Install

```bash
npx dealdesk@latest onboard
```
