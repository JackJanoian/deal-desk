# v2026.527.0

> Released: 2026-05-27

First public release of **DealDesk** — AI-powered deal sourcing for private equity teams, distributed on npm.

## Highlights

- **Install from npm** — Get started with `npx dealdesk@latest onboard`. The CLI bootstraps a local DealDesk instance with an embedded database, no cloud services required.
- **Local-first by default** — Runs entirely on your machine. Telemetry is opt-in.
- **Agent adapters included** — Built-in adapters for Claude, Codex, Cursor (local and cloud), Gemini, OpenCode, and more, so you can drive deal-sourcing workflows with the agent runtime of your choice.
- **Plugin SDK** — Extend DealDesk with scoped database namespaces, managed agents and routines, scoped APIs, and reusable UI building blocks via `@dealdesk/create-dealdesk-plugin`.

## Install

```bash
npx dealdesk@latest onboard
```
