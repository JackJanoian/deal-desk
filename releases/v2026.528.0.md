# v2026.528.0

> Released: 2026-05-28

## Fixes

- **Server startup no longer crashes on `type "dd_email_status" already exists`** — The migration reconciler now handles `CREATE TYPE ... AS ENUM`, leading SQL comment blocks, and the cascade case where an early migration's table/index/constraint is gone because a later migration intentionally dropped it. The reconciler distinguishes "user manually removed it, must re-run" from "later migration intentionally dropped it, safe to skip" by scanning for a matching `DROP` in later migration content.
